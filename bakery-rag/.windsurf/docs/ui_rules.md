# UI Guidelines

## Primary Color
- The primary color for the UI is **#F4A261** (orange/peach).

## RTL Support
- The system must fully support **Right-to-Left (RTL)** layout.
- All UI components, including text, icons, and navigation, should be mirrored appropriately for RTL.

## Buttons
- All buttons must have **rounded corners** for a consistent and friendly look.
- Button styles should respect the primary color and RTL layout.

## Additional Notes
- Ensure accessibility standards are met.
- Use consistent spacing and typography throughout the UI.
